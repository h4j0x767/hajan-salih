What is JavaScript and Why is it Important?

A lightweight, interpreted language for creating network applications.

Key Features

  • گونجای و دینامیکیە
  • پشتەڤانیا بەرنامەسازیا فەنکشنەل دکەت
  • رێ ددەت مالپەڕێن کارلێکەر چێببن
  • Versatile and dynamic
  • Supports functional programming
  • Enables interactive web pages

Store data. Use var, let, or const to declare.

Control Flow: Making Decisions with JavaScript

دەستورێن مەرجدار

Conditional Statements

if, else if, و else رێ ددەن جێبەجێکرنا مەرجدار ل سەر بنگەهێ دەربڕینان

if, else if, and else allow conditional execution based on expressions.

گەڕین

Loops

گەڕین ل سەر پارچەیەکا کۆدێ دووبارە دبن. for و while بۆ ئەرکێن دووبارە بکاربینە

Loops iterate over a block of code. Use for and while for repetitive tasks.

دەستورێن switch

Switch Statements

switch رێکەکا پر-رێ یا ڤەقەتاندنێ پێشکێش دکەت. ب کارامەیی گەلەک مەرجان بڕێڤە دبەت

switch provides a multi-way branch. Efficiently handles multiple conditions.

کارکرن دگەل DOM

Working with the DOM

مۆدێلا ئۆبجێکتا دۆکیومێنتێ

Document Object Model

DOM پارچەیێن HTML وەک ئۆبجێکت نیشان ددەت. رێ ددەت جاڤا سکریپت ناڤەرۆکێ بگوهۆڕیت

The DOM represents HTML elements as objects. It enables JavaScript to modify content.

گەهشتنا پارچەیان

Accessing Elements

مێتودێن وەک getElementById و querySelector بکاربینە بۆ هەلبژارتنا پارچەیێن دیارکری

Use methods like getElementById and querySelector to select specific elements.

گوهۆڕینا ناڤەرۆکێ

Modifying Content

تێکست، تایبەتمەندی و ستایلان بگوهۆڕە. ناڤەرۆکا مالپەڕێ ب شێوەیەکێ دینامیکی نوی بکە

Change text, attributes, and styles. Dynamically update web page content.

کریارێن نەهەڤکات

Asynchronous Operations

ئەرکان بێی بلۆککرن بڕێڤە ببە. بەرسڤدانێ و ئەزموونا بکارئینەری باشتر بکە

Handle tasks without blocking. Improve responsiveness and user experience.

Promises

Promises

ئەنجامدانا (یان سەرنەکەفتنا) کریارەکا نەهەڤکات نیشان ددەت

Represent the eventual completion (or failure) of an async operation.

Async/Await

Async/Await

کۆدا نەهەڤکات سادە دکەت. کۆدەکا زەلالتر و خویندبارتر دنڤیسیت

Simplify asynchronous code. Write cleaner and more readable code.
